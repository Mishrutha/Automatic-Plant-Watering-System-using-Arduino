# Automatic Plant Watering System Using Arduino Uno

## Overview
This project automates plant irrigation by monitoring soil moisture and delivering water automatically using Arduino Uno. It ensures healthy plant growth and efficient water usage.

## Project Contents
- **code/**: Arduino sketches for automation
- **circuit_diagram/**: Wiring diagram and circuit setup
- **report/**: Detailed project report
- **output/**: Images showing working system

## Features
- Automatic watering based on soil moisture readings
- Real-time control using Arduino Uno
- Easy to set up and maintain

## Technologies Used
- Arduino Uno
- Soil Moisture Sensor
- Water Pump / Relay Module
